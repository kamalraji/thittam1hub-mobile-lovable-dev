# Product Overview

**Thittam1Hub** (also referred to as "Dreamflow" in some contexts) is a Flutter-based event management and community platform with social impact features.

## Core Features

- **Event Discovery**: Browse, search, and filter events by category, mode (online/offline/hybrid), and status
- **Event Management**: Full event lifecycle with ticketing, registration, and capacity management
- **Impact Hub**: Community engagement features including Circles (group discussions), Spaces (audio rooms), and social profiles
- **Chat System**: Workspace channels and direct messaging with file attachments
- **User Profiles**: Customizable portfolios with QR codes, social links, and profile completeness tracking
- **Authentication**: Supabase-powered auth with sign-in/sign-up flows

## Target Platforms

- Android
- iOS
- Web
- Linux Desktop

## Key User Flows

1. Browse events → View details → Register with ticket selection
2. Join Impact Hub → Participate in Circles/Spaces → Connect with profiles
3. Chat in workspace channels or send direct messages
4. Manage personal profile and view QR code for networking
